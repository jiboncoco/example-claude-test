# example-claude-test

An Angular 17 starter app with a login page, built with Tailwind CSS.

## Tech Stack

- [Angular 17](https://angular.dev) — standalone components, new control flow syntax (`@if`, `@for`)
- [Tailwind CSS 3](https://tailwindcss.com) — utility-first styling
- TypeScript 5.4

## Getting Started

```bash
npm install
npm start
```

App runs at `http://localhost:4200`.

## Build

```bash
npm run build
```

Output goes to `dist/`.

## Project Structure

```
src/

│   ├── pages/

```
